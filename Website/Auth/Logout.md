# Logout

Used to Logout.

**URL** : `/v1/Website/Auth/Logout`

**Method** : `GET`

**Auth required** : YES

**Data constraints**

```json
no data required
```

**Data example**

```json
no data required
```

## Success Response

**Code** : `200 OK`

**Output Constraints**

```json
{
    "status": "Success"
}
```

**Output example**

```json
{
    "status": "Success"
}
```

## Error Response

**Condition** : no error

**Code** : `no error`

**Content** :

```json
no error
```
